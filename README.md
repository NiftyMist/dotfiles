# dotfiles

Configuration files 

# Installation

Clone into home directory.

# Requirements

- `zsh`
  - `zsh-autosuggestions`
  - `zsh-syntax-highlighting`
- powerline fonts or nerd fonts
  - `JetBrainsMonoNL Nerd Font` prefered
- `stow`
- homebrew
  - bat
  - fzf
  - docker-completion
  - lazygit
  - neovim
  - oh-my-posh
  - sshpass
  - stow
  - tmux
  - starship
